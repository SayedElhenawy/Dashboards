# Data Jobs Dashboard w/ Power Bi

![Dashboard](/Images/Screenshot%202026-03-02%20at%203.59.36 AM.png)

## Introduction

This Dashboard is created for **job seekers that want to compare jobs data (Salaries, common jobs, Skills & Countries)** using real world dataset of 2024 data science job postings.

## Skills Showcased

- **Simplicity** : 
The dashboard is intuitive, easy to use, contains all the useful info that at a first glance anybody would be looking for.

- **Visualizations** :
Numbers are easily understood once looked at from the dashboard.

- **Parameters** : Use of Parameters with bar charts to provide easy access to multiple types or relations within the same space to provide all accurate data at ease.

- **Slicers** : To dynamically filter the dashboard between Country, Skill, Job Title, Company, and the main job the use is focused on.

- **Cards** : To easily see data regarding salary and job counts, selected job (filtered).

- **Charts** : Use of Pie & Donut charts to be easy on the eye while providing the necessary info needed in a fun and eye fulfilling way.
