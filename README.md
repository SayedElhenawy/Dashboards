# Data Jobs Dashboard w/ Power Bi

![Dashboard](/Jobs%20Dashboard/Images/Screenshot%202026-03-02%20at%203.59.36 AM.png)

## Introduction

This Dashboard is created for **job seekers that want to compare jobs data (Salaries, common jobs, Skills & Countries)** using real world dataset of 2024 data science job postings.

## Skills Showcased

- **Simplicity** : 
The dashboard is intuitive, easy to use, contains all the useful info that at a first glance anybody would be looking for.

- **Visualizations** :
Numbers are easily understood once looked at from the dashboard.

- **Parameters** : Use of Parameters with bar charts to provide easy access to multiple types or relations within the same space to provide all accurate data at ease.

- **Slicers** : To dynamically filter the dashboard between Country, Skill, Job Title, Company, and the main job the use is focused on.

- **Cards** : To easily see data regarding salary and job counts, selected job (filtered).

- **Charts** : Use of Pie & Donut charts to be easy on the eye while providing the necessary info needed in a fun and eye fulfilling way.

## Conclusion

I have been working with excel & google sheets for a long time, I have made reports, automated tempelates, visualized data and made pivote table, yet they don't compare to how easily a dashboard like this can be made in power bi especially with how large the data is.


# Call Center Dashboard

![Dashboard](/Call%20Center%20Dashboard/Images/Screenshot%202026-03-03%20at%2011.27.25 PM.png)

## Introduction

As for this one it's created for **managers at a call center floor montiroing data such as Csat % (Customer Satisfaction %), RR % (Resolution Rate %), Calls offered and abandoned...etc** This one highlights most valuable numbers that need to be monitored on daily basis while including other info that help give insights like topics going in, % of Answered vs Resolved which can help Managers make accurate decisions leaving no room for confusion.

## Skills Showcased

- **Highlights** :
Most important numbers that will be discussed on cadence everyday are at the top, using cards which make it easy.

- **Charts** : Use of Pie charts provide and easy on the eye experience while containing info that are of well use to the managers discussing them.

- **ratios** : Use of ratios that that give insights on what topics "Maybe Problems" might be driving a spike in numbers of calls.

- **Simplicity** : Nothing about this. dashboard is that hard to interact with, leaves no room for wasting time.

## Conclusion : 
For work especially at a large field practicality, simplicity and ease of use are what matter the most, easy for everybody to understand at a glance, get used to and possibly manage.
